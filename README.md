## 🚀 설치 방법

### 1. Node.js 설치
- Node.js 14 이상이 필요합니다
- https://nodejs.org/ 에서 다운로드

### 2. 패키지 설치
```bash
cd mzsol-homepage
npm install
```

## 🌐 80번 포트로 실행

### Mac / Linux
```bash
sudo npm start
```

### Windows
1. 명령 프롬프트를 **관리자 권한**으로 실행
2. 프로젝트 폴더로 이동
3. 다음 명령어 실행:
```bash
npm run start:windows
```

### 접속
브라우저에서 http://localhost:80 으로 접속

⚠️ **주의사항:**
- 80번 포트는 시스템 포트로 관리자 권한이 필요합니다
- 다른 프로그램이 80번 포트를 사용 중이면 충돌이 발생할 수 있습니다
- 개발 시에는 기본 포트(3000)를 사용하고, 배포 시에만 80번 포트를 사용하는 것을 권장합니다
