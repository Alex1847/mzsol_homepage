🔢 IoT 카운터 장비개발

현장 설비의 생산 수량·동작 횟수·이벤트 발생을 실시간으로 수집하고
디지털 표시와 원격 모니터링까지 연결하는 스마트 카운터 장비

실시간 수량 카운팅 디지털 LED 표시 센서 입력 연동 무선 데이터 전송 알람·임계값 관리 Web 대시보드 연계

현장형 디지털 카운터

설비의 펄스, 근접센서, 포토센서, 버튼 입력을 받아 카운트 값을 즉시 표시하고 서버로 전송합니다. 작업자는 현장에서 숫자를 바로 확인하고, 관리자는 웹 화면에서 누적 수량과 이상 이벤트를 함께 확인할 수 있습니다.

통신 연동 Wi-Fi·RS485·Gateway 확장
관리 기능 목표 수량·알람·이력 조회

센서보드 기반 데이터 수집

현장 센서보드에서 온도, 습도, 압력, 미세먼지 데이터를 수집하고 MQTT 서버를 통해 Web 서버로 전달합니다. 관리자는 PC에서 이력과 트렌드를 저장·분석하고, 필요 시 휴대폰으로 실시간 알림을 받을 수 있습니다.

수집 대상 온습도·압력·미세먼지
중계 방식 MQTT 서버 기반 메시지 전송
확인 채널 Web 화면·휴대폰 알림
저장 기능 PC 이력·트렌드 데이터 저장
MQTT 센서 데이터를 안정적으로 중계하는 메시지 서버
Web 브라우저 기반 실시간 모니터링 화면
Trend PC 이력 저장과 추이 분석으로 현장 변화 추적
