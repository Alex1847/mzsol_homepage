**index.js 파일 설명:**

1. **`import React from 'react'`**
   - React 라이브러리 임포트

2. **`import ReactDOM from 'react-dom/client'`**
   - React 18의 새로운 클라이언트 렌더링 API

3. **`import './index.css'`**
   - 글로벌 CSS 스타일 임포트

4. **`import App from './App'`**
   - 메인 App 컴포넌트 임포트

5. **`ReactDOM.createRoot()`**
   - React 18의 새로운 루트 생성 방식
   - `document.getElementById('root')`는 public/index.html의 `<div id="root"></div>`를 찾음

6. **`<React.StrictMode>`**
   - 개발 모드에서 잠재적인 문제를 감지
   - 프로덕션 빌드에서는 영향 없음

**작동 흐름:**
```
public/index.html
    ↓
<div id="root"></div>에 React 앱 마운트
    ↓
src/index.js (진입점)
    ↓
src/App.js (메인 컴포넌트 렌더링)
이 파일은 React 앱의 진입점(Entry Point)이며, 프로젝트 시작 시 가장 먼저 실행됩니다!Aindex.css파일을보여줘다운로드되는 index.css 파일 내용입니다:

**index.css 파일 설명:**

1. **`body { margin: 0; }`**
   - 브라우저 기본 여백 제거
   - 페이지가 화면 전체를 사용하도록 설정

2. **`font-family`**
   - 시스템 네이티브 폰트 우선 사용
   - `-apple-system`: macOS/iOS용
   - `BlinkMacSystemFont`: Chrome on macOS용
   - `'Segoe UI'`: Windows용
   - `'Roboto'`: Android용
   - 폴백(fallback) 폰트들이 순서대로 나열됨

3. **`-webkit-font-smoothing: antialiased`**
   - macOS/iOS에서 폰트를 더 부드럽게 렌더링
   - 텍스트 가독성 향상

4. **`-moz-osx-font-smoothing: grayscale`**
   - Firefox에서 폰트 렌더링 최적화

5. **`code { font-family: ... }`**
   - 코드 블록용 모노스페이스(고정폭) 폰트 설정
   - 개발자 도구나 코드 표시 시 사용

**역할:**
- 이 파일은 **전역(Global) CSS**로, 앱 전체에 적용됩니다
- `App.css`는 컴포넌트별 스타일링을 담당합니다

```
